# Telegram Username Bot

Бот для генерации и проверки 5-значных юзернеймов в Telegram.

## Возможности

- Генерация красивых 5-значных юзернеймов
- Проверка доступности через Fragment API и Telegram API
- Автоматический поиск свободного юзернейма
- Удобный интерфейс с кнопками

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте файл `.env` с переменной `BOT_TOKEN`
4. Запустите: `python bot.py`
